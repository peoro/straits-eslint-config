
# @straits/eslint-config

> straits' ESLint config

## Installation

```bash
npm install --save-dev @straits/eslint-config
```

## Usage

Set your `.eslintrc.*`'s `extends` field to `"@straits/eslint-config"`:

```json
{
	"extends": "@straits/eslint-config"
}
```

That's it, ESLint will now use this config.
